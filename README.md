# 个人品牌IP形象生成器

根据个人定位生成统一的IP形象设计，支持多场景延展。

## 🎯 功能特点

- **定位提炼**：从个人定位提取IP性格关键词、视觉元素
- **5大核心设定**：基础形象、服装配饰、配色方案、性格表情、场景延展
- **一致性保障**：确保IP在不同场景下保持视觉统一
- **多场景输出**：头像、表情包、海报元素、周边延展

## 📥 输入

- 个人定位（如：AI产品经理、健身教练、心理咨询师）
- 性格关键词（3-5个）
- 目标受众
- 偏好风格（简约几何/手绘风/二次元/商务简约）
- 应用场景（头像/表情包/海报/周边）
- 参考元素（可选）

## 📤 输出

- IP全身/半身形象设计
- 配色方案
- 3-5个标志性表情
- 多场景延展示例

## 🚀 使用示例

**输入：**

> 定位：AI产品经理
> 性格关键词：理性、专业、有想法、略带幽默
> 受众：AI从业者、产品经理
> 风格：商务简约
> 场景：头像、表情包、公众号插图

**输出：**

- 基础形象：戴眼镜的简洁卡通人物，职业装
- 配色：科技蓝+深灰+白色
- 表情：思考状、认真状、调侃状、疑惑状
- 场景延展：头像版、表情包系列、海报元素版
